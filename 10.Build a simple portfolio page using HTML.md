# 🎬 10. Build a Simple Portfolio Page (Project)

## 🚀 Project Goal

Create a **basic personal portfolio page** using HTML.

👉 You will learn how to combine:

* Headings
* Images
* Links
* Sections

---

## 🧠 What is a Portfolio?

A portfolio is a **personal website** that shows:

* Who you are 👤
* What you do 💻
* Your work/projects 🚀

---

## 🧱 Project Structure

We will build:

* Header (Name & title)
* About section
* Projects section
* Contact section

---

## 📦 Complete Code

```html id="7xk9ab"
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Portfolio</title>
</head>
<body>

  <!-- Header -->
  <header>
    <h1>Aviranjan</h1>
    <p>Web Developer</p>
  </header>

  <!-- About Section -->
  <section>
    <h2>About Me</h2>
    <img src="profile.jpg" alt="Profile photo of Aviranjan" width="150">
    <p>I am learning web development and building real projects.</p>
  </section>

  <!-- Projects Section -->
  <section>
    <h2>My Projects</h2>
    <ul>
      <li>HTML Website</li>
      <li>Portfolio Page</li>
    </ul>
  </section>

  <!-- Contact Section -->
  <section>
    <h2>Contact</h2>
    <p>Email: example@email.com</p>
    <p>
      <a href="https://github.com/">GitHub</a>
    </p>
  </section>

  <!-- Footer -->
  <footer>
    <p>© 2026 Aviranjan</p>
  </footer>

</body>
</html>
```

---

## 🔍 What You Used

| Feature  | Used In Project |
| -------- | --------------- |
| Headings | `<h1>`, `<h2>`  |
| Image    | `<img>`         |
| Links    | `<a>`           |
| Sections | `<section>`     |

---

## 🌍 How It Works

* `<header>` → Your introduction
* `<section>` → Different parts of page
* `<img>` → Profile picture
* `<a>` → External link (GitHub)

👉 This is your **first real website structure**
