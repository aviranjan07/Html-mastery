# 🌐 1. Introduction to HTML

## 🚀 What is HTML?

**HTML (HyperText Markup Language)** is the standard language used to create web pages.

👉 It tells the browser:

* What content to show
* How to structure that content

Think of HTML as the **skeleton of a website** 🦴

---

## 🧠 Real-Life Analogy

* HTML → Skeleton (structure)
* CSS → Skin & design (looks)
* JavaScript → Brain (behavior)

👉 Without HTML, a website cannot exist.

---

## 🌍 How Websites Actually Work

When you open a website:

1. You type a URL (like google.com)
2. Browser sends a request to a server
3. Server sends back HTML
4. Browser reads & displays it

👉 Browser converts HTML into something called a **DOM (Document Object Model)**

---

## 📦 Basic Structure of an HTML Document

Every HTML page follows a standard structure:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first HTML page</p>
  </body>
</html>
```

---

## 🔍 Explanation of Each Part

### 1. `<!DOCTYPE html>`

* Tells browser this is an HTML5 document

---

### 2. `<html>`

* Root element (everything is inside this)

---

### 3. `<head>`

* Contains:

  * Title
  * Meta info
  * Links to CSS

👉 Not visible on the webpage

---

### 4. `<title>`

* Shows on browser tab

---

### 5. `<body>`

* Contains all visible content:

  * Text
  * Images
  * Buttons
  * Links

---

## 🧱 HTML Tags Basics

HTML uses **tags** to define elements:

```html
<h1>This is a heading</h1>
<p>This is a paragraph</p>
```

### Structure:

```
<tag> Content </tag>
```

---

## ⚡ Important Points

✔ HTML is **not a programming language**
✔ It is a **markup language**
✔ It defines **structure, not style**
✔ Browser reads HTML from **top to bottom**

---

## 🧪 Simple Example

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Demo</title>
  </head>
  <body>
    <h1>Welcome</h1>
    <p>Learning HTML is fun!</p>
  </body>
</html>
```

👉 Output:

* A heading: "Welcome"
* A paragraph: "Learning HTML is fun!"
